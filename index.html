List Types in HTML

There are 3 types of Lists

  1. Ordered Lists
  2. Unordered Lists
  3. Description Lists
Ordered List in HTML

Till now we were using nbsp to add space and make bullet points but now we can do it automatically by using tags. i.e. to start an ordered list we have to use <ol> at the start of the list and </ol> at the end of the list. then to add bullet points we need to use <li> content of bullet point </li>

  1. Point A
  2. Point B
  3. Point C
Unordered List in HTML

To start an unordered list we have to use <ul> at the start of the list and </ul> at the end of the list. then to add bullet points we need to use <li> content of bullet point </li>. Here you will notice that instead of numbering, bullets are used.

  • Point A
  • Point B
  • Point C
Description List in HTML

To start description list we have to use <dl> at the start of the list and </dl> at the end of the list. then to add bullet points we need to use <dt> description Term </dt> and <dd> description Details </dd>. Here you will notice that instead of numbering and bullets we can add discription of each point. Discription term is to the left and description details are indented a little bit to the right

Point A
Description of Point A
Point B
Description of Point B
Point C
Description of Point C

Adding Links in HTML

There are 3 types of links:

  1. Absolute Reference
  2. Relative Reference
  3. Internal Reference

To link anything we use anchor tag. Lets link the youtube tutorial I am using to learn HTML. For this is have used anchor tag at the start and end. in the starting anchor tag I have added the link of this video in href i.e. <a href="https://www.youtube.com/">HTML</a> and you are good to go.

This is an absolute reference because we are giving the full address of the web page to this page.

There is another term relative reference which is for something that is on the same server/folder where we have our page.

The 3rd one is to link a specific part/area/portion of the page to something also called as internal reference. for this 1st of all we have created different sections and given those sections an id. then we have used anchor tag and in href we have given the id name of the section with a # in the start. it will look like this: <a href="#section_id"> text </a>, if we only use # in href and do not write any section id it will automatically go to the top of the page.

If you want to make a download link i.e. if you click on the link it will start downloading the linked file. you have to make an anchor tag like this: <a href="name of file" download> text </a>

If you want to make a Contact me by email link i.e. if you click on the link it will send an email on your email account. you have to make an anchor tag like this: <a href="mailto:random@email.com"> text </a>

If you want to make a Contact me by Phone No. link i.e. if you click on the link it will directly call you on your mobile no. you have to make an anchor tag like this: <a href="tel:+921234567890"> text </a>

If you want to make a link that will open in new tab, you have to make an anchor tag like this: <a href="link" target = "_blank" > text </a>

And finally if you want to make a link by which you want to come back on the home page, you have to make an anchor tag like this: <a href="/" > text </a>
